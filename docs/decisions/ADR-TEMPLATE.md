# Architecture Decision Record Template

## Purpose

Provide a standard format for documenting major technical and architectural decisions in NEWAX Core.

Architecture Decision Records help NEWAX preserve long-term maintainability, clarity, accountability, and engineering discipline as the platform evolves.

Use this template for important architecture decisions only, not every small code change.
